# fractal-noise
tandalone MATLAB toolbox for generating, shaping, estimating, and diagnosing 1/f^alpha (fractal/colored) noise. noiseXu extends past dsp.ColoredNoise and fractalcoef's alpha=[-2,2] limit to [-2,7] via Xu (2019) fractional differencing. Includes IRASA estimation with independent cross-validation and a Lorentzian-knee diagnostic. No dependencies.
